# Jaskirat Singh — Portfolio

Cyberpunk glitch portfolio for Jaskirat Singh, first-year UI/UX design student.

## Run locally

```bash
npm install
npm start
```

Then open http://localhost:3004

Admin panel: http://localhost:3004/admin (default password: `jaskirat2026`).

## Features

- Projects with multi-image upload + dedicated project pages
- Editable profile, bio, personal touch, and goals
- Semester I curriculum with subjects + faculty
- Skills with level bars
- Three photo slots (hero portrait, about portrait, brand mark)
- Editable contact / social links (Email, Instagram, Twitter, LinkedIn)
- Password-protected admin

## Tech

Express + multer, vanilla HTML/CSS/JS frontend. Data persisted in `data/` (gitignored).
